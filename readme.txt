Что стоит попоравить к финальному макету:

блок 3:

Стили сброса браузерных стилей можно вынести в normalize/reset

Блокам нельзя задавать внешнюю геометрию

В мобильной версии прокрутку меню стоит сделать и за левый край экрана



блок 4:

Стили кнопок при наведении должны плавно меняться

Не стоит transition применять ко всем свойствам, по возможности нужно применять к тем свойствам, которые должны плавно изменяться, для предотвращения создания излишней нагрузки

Стоит сделать резиновым по вертикали и чтобы полоса прокрутки появлялась только когда заканчивается свободное пространство

Стили для hover не надо задавать отдельно через Модификатор, они и так отдельно задаются через псевдокласс

Телефон и адрес электронной почты можно сделать ссылкой с корректным href

Всё левое меню можно поместить в aside
